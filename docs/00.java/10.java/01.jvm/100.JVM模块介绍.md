## JVM 总体图 

学习JVM引导：
* 首先了解JVM模块的组成，由4个组成部分 `类加载器`、`运行时数据区`、`执行引擎`、`本地方法接口`
* 了解JVM整个运行机制，了解类加载机制
* 了解运行时数据区的重要组成部分，`PC程序计数器`、`虚拟机栈`、`堆区`、`元数据区`、`本地方法栈`
