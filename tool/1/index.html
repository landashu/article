vscode ide 配置

# java 相关

vscode 主要以插件和配置 setting.json 文件为主，java 就下载主流的 Extension Pack for Java 插件，他包含了 maven 插件，java 支持等。

再去配置 setting.json 文件，添加如下内容：

{
    "java.jdt.ls.java.home": "D:\\tools\\java\\java25",
    "files.exclude":{
        ".idea": true
    },
    "maven.executable.path": "D:\\tools\\maven-3.8.4\\bin\\mvn.cmd",
    "java.configuration.maven.userSettings": "D:\\tools\\maven-3.8.4\\conf\\settings.xml"
}
