# 配置 java

  1. 下载插件
    Language Support For Java 让 vscode 支持 java 语言， Extension Pack for Java 扩展包， Java Imports 自动导入包

  2. 配置 JDK路径 ，在设置，用户中搜索 java.home ，然后在 settings.json 中配置："java.jdt.ls.java.home": "D:\tool\java\java17",

  1. 配置 launch.json 文件，这个是个性化配置，查看官网后自行配置
