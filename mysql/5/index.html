MySQL 8安装

建议看完文章再装，不一定适合所有。

下载地址 (opens new window)；这个是官方文档，告诉你如何再 Linux 上安装 mysql (opens new window)

个人不太喜欢用 yum 安装，就是 yum 安装找不到路径，对于我这种半吊子，我还是喜欢 tar 的文件。

# 下载合适的版本

我这里下载的 MySQL 8

wget https://cdn.mysql.com/archives/mysql-8.0/mysql-8.0.20-el7-x86_64.tar.gz
1

解压缩

tar -xvf mysql-8.0.20-el7-x86_64.tar.gz
1

libaio 如果系统上尚不存在 该库，则可能需要安装该库

wget http://mirror.centos.org/centos/6/os/x86_64/Packages/libaio-0.3.107-10.el6.x86_64.rpm
rpm -ivh libaio-0.3.107-10.el6.x86_64.rpm
1
2

添加用户组

groupadd mysql
useradd -g mysql mysql
1
2

解压后记得改文件名字，要不然源文件太长， 创建 mysql 文件，并赋权

cd mysql8
mkdir mysql-files
chown mysql:mysql mysql-files
chmod 750 mysql-files
1
2
3
4

查看一下文件的权限，使用命令 ll

drwxr-x---.  2 mysql mysql     25 Jul 16 13:56 mysql-files
1

drwxr-x---. 一共 10 个字符，下面讲解下：

  • d 表示目录，如果是 - 表示是一个普通文件。剩余的 9 个字符，分成 3 组，每组 3 个字符，分别表示 owner/group/others 的 rwx 权限；
  • owner 表示拥有者，可以看到拥有者是 mysql 用户
  • group 表示组，除了 mysql 这个人的 同一个 MySQL 组拥有的权力
  • others 就是其他人了，啥权限也没有。

  • 在 初始化的时候填写 basedir

# 远程连接

接下来就是添加一个用户，然后该用户可以远程访问 mysql。以下的所有报错我就不在说了，只告诉正确的操作，执行问之后大家就可以连接了。

mysql> use mysql;
Reading table information for completion of table and column names
You can turn off this feature to get a quicker startup with -A

Database changed
1
2
3
4
5
mysql> flush privileges;
Query OK, 0 rows affected (0.00 sec)
1
2
mysql> create user 'dev'@'%' identified by 'mysql@dev.2020';
Query OK, 0 rows affected (0.00 sec)
1
2
mysql> grant all on *.* to 'dev'@'%';
Query OK, 0 rows affected (0.00 sec)
1
2

修改密码规则，要不然我们的工具是连接不上的。

mysql> ALTER USER 'dev'@'%' IDENTIFIED BY 'mysql@dev.2020' PASSWORD expire never; 
Query OK, 0 rows affected (0.01 sec)
1
2

修改密码

mysql> ALTER USER 'dev'@'%' IDENTIFIED WITH mysql_native_password BY 'mysql@dev.2020';
Query OK, 0 rows affected (0.00 sec)
1
2

修改 root 密码，不要忘记

alter user 'root'@'localhost' identified by 'Root@123456';
1

刷新

mysql> flush privileges;
Query OK, 0 rows affected (0.05 sec)
1
2
